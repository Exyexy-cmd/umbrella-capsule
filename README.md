# ☂️ Umbrella Capsule

Projeto de landing page desenvolvido com HTML e CSS puro, simulando um e-commerce de guarda-chuvas premium.

## 🚀 Funcionalidades
- Página inicial (Hero)
- Página "Sobre"
- Página de produtos
- Navegação entre páginas

## 🎯 Objetivo
Treinar estrutura de layout, estilização com CSS e organização de páginas.

## 🛠️ Tecnologias
- HTML5
- CSS3
